1 Numeric

Numeric类用来表示数字，其中又包含了Integer，Float，Complex，BigDecimal，Rational共 5 个子类。

在日常开发中，Integer与Float类使用的较多。

1.1 Integer

当然，Ruby中也存在着一些意外的规则。
不同数据类型的溢出规则
Interger在运算后会对大数进行转化（Bignum），因此Integer不会产生溢出。但这一点在Float上却并不适用，当数据超出范围后它会溢出为Infinity/-Infinity/0三种情况。

不同数据类型的除法
在Integer的除法运算中，当除数为 0 时，会抛出ZeroDivisionError异常（我通常使用这种方式来刻意触发异常）、而在Float的除法运算中，除数为 0 时会返回Infintiy/-Infintiy。当然，这里也有一个例外，当发生0.0/0.0的运算时，会返回NaN（not a number）
